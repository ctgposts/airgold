# Contributing to Dhaka Airways Ticketing System

Thank you for your interest in contributing! This document provides guidelines for contributing to the project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/dhaka-airways-system.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`

## Development Setup

No build process required! Simply:
1. Open `index.html` in a web browser, or
2. Use a local server: `npx serve .`

## Code Style

- Use consistent indentation (2 spaces)
- Follow existing code patterns
- Add comments for complex logic
- Keep functions small and focused

## Submitting Changes

1. Ensure your code works in latest Chrome, Firefox, and Safari
2. Test all user roles (Admin, Manager, Staff)
3. Commit your changes with clear messages
4. Push to your fork
5. Submit a Pull Request with description of changes

## Reporting Issues

- Use GitHub Issues
- Include browser and version
- Provide steps to reproduce
- Include screenshots if applicable

## Questions?

Open an issue for questions or discussions.

Thank you for contributing!
